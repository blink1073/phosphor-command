add

  • add(commands: ICommand[]): IDisposable
    • Defined in index.ts:368

    Add commands to the registry.

    Parameters

    • commands: ICommand[]

      The commands to add to the registry.

    Returns IDisposable

    A disposable which will remove the added commands.

    Notes

    If the id for a command is already registered, a warning will be logged and that specific command will be ignored.

    The id of a command must be constant. Mutating the command id while the command is registered will result in undefined behavior.
